# Workstation

## Install Ansible
```sh
sudo apt update
sudo apt upgrade
sudo apt install ansible
```

## Run playbook

```sh
ansible-pull -U https://github.com/licvido/workstation.git
```

## Setup git

```sh
git config --global user.name "Filip Mikovcak"
git config --global user.email "fmi@pop-os"
```
